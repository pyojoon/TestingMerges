Initial Commit
cd - Change Directory
Mkdir - Make Directory
git branch Akan - Creates a new Branch
git checkout Akan - Switches to Akan's Branch
touch AkanReadMe.txt - Creates a text file called AkanReadMe
git status - Displays the paths that have differences between the index file and the current HEAD commit, paths 
that have differences between the working tree and the index file and the paths in the working directory that
are not tracked by Git.
git add . - Updates the index using all the current content found in the working tree, to prepare the content
staged for the next commit.
git commit -m "Akan's First Commit" - Commit the changed files to the specified branch and renames it as Akan's
first Commit.
